Erfolg class

Repräsentiert einen Erfolg oder eine Auszeichnung in einer Anwendung.

Ein Erfolg besteht aus einem Namen, einer Beschreibung und einer Schwierigkeitsstufe (z. B. Bronze, Silber, Gold).
