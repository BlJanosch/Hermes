Sammelkarte

Sammelkarte class

Repräsentiert eine Sammelkarte mit Bild, Schwierigkeitsgrad und weiteren Eigenschaften.

Eine Sammelkarte enthält Metadaten wie Name, Bildpfad, Schwierigkeitsstufe, Höhe des Berges und das Erstellungs- oder Funddatum.
