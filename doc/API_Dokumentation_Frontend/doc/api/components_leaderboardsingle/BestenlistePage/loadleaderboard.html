Future<List<Map<String, dynamic>>> loadleaderboard()

Lädt die Bestenliste vom Server.

Nutzt die aktuell gespeicherte User-ID und die angegebene Kategorie als Filter. Gibt eine Liste von Maps zurück, die die Einträge der Bestenliste darstellen.

Implementation

Future<List<Map<String, dynamic>>> loadleaderboard() async {
  final prefs = await SharedPreferences.getInstance();
  int? id = prefs.getInt('id');
  final url = Uri.parse('http://$serverIP:8080/user/bestenliste?userID=$id&filterDB=$category');
  final response = await http.get(url);
  final result = json.decode(response.body);
  print(result);

  return List<Map<String, dynamic>>.from(result);
}
